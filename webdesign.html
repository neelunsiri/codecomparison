Web Design

Background Information

  • Web Design consists of HTML, CSS and JavaScript
  • HTML, which means HyperText Markup Language, is like the skeleton of a webpage, and states what the webpage has on it.
  • CSS, which means Cascading Style Sheets, is like the skin of a webpage. It styles the webpage to make every element look the way the developer intends.
  • JavaScript, abbreviated to JS, gives a webpage functionality, and makes it interactive.

Example Program in HTML

<!DOCTYPE html>
<html>
 <head>
  <title>Hello World!<title>
 </head>
 <body>
  <p>Hello World!</p>
 </body>
</html>

This program would make the title of the webpage "Hello World!", and print "Hello World!" on the page.

Example Program in CSS

body {
 background-color:green;
}

p {
 font-family:Impact;
 font-size:26px;
}

This program, when linked to an HTML document, will make the background colour of the page green, and the font of every paragraph on the page, except declared exceptions, have text that is 26 font-size Impact font.

Example Program in JavaScript

function typeThis() {
 var whatToSay=document.getElementById("input").value;
 document.getElementById("paragraphText").innerHTML=whatToSay;
}

This JavaScript program, when named script,js and linked to an HTML program that looks something like the following, takes whatever the user types in the text box, and, when the button is clicked, displays it just below the text box and button.

<!DOCTYPE html>
<html>
 <head>
  <title>My JavaScript Program!</title>
  <script src="script.js"></script>
 </head>
 <body>
  <input type="text" id="input"></input>
  <button onclick="typeThis()">Type it!</button>
  <br>
  <p id="paragraphText">Please type something in the box and click "Type it".</p>
 </body>
</html>

Pros and Cons

Pros

  • Easy to Learn
  • Almost anything that people make/do will be improved and/or enhanced with an Internet website.
  • It is much cheaper to write your own HTML, CSS, and JavaScript that to hire a professional, or to buy professional software.

Cons

  • Although writing a simple website requires very little knowledge, writing a more advanced website, requires more advanced skills like design skills.
  • It is also neccessary to learn a server-side language like PHP to enhance the capabilities of a website.
