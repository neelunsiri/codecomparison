Java

Background Information

  • Java is one of the most widely-used programming languages.
  • It is a very powerful language, and also relatively simple to learn.
  • Java is quite a universal language, and is used on computers, TVs, CD/DVD players, Cars, Irrigation Systems, and many more, including even fridges!
  • It can be used to make a wide range of programs, notably universal games.
  • It is not the same thing or similar as JavaScript, other than the fact that they are both Object-Oriented Programming Languages.

Example Program in Java

public class HelloWorld{

 public static void main(String []args){
  System.out.println("Hello World");
 }
}

This program, when executed, prints "Hello World!".

Pros and Cons

Pros

  • Very universal, and runs on almost every device.
  • Easy to learn the basics and write a simple program.
  • A very powerful programming language.
  • Can be used as both an Object-Oriented programming language and a Server-Side Scripting Language.

Cons

  • Like most other programming languages, it is quite difficult to write a more advanced program.
  • It is commonly confused with JavaScript (although this isn't really a problem for programming).
