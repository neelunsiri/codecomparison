Web Design

Background Information

  • Web Design consists of HTML, CSS and JavaScript
  • HTML, which means HyperText Markup Language, is like the skeleton of a webpage, and states what the webpage has on it.
  • CSS, which means Cascading Style Sheets, is like the skin of a webpage. It styles the webpage to make every element look the way the developer intends.
  • JavaScript, abbreviated to JS, gives a webpage functionality, and makes it interactive.

Example Program in HTML

<!DOCTYPE html>
<html>
 <head>
  <title>Hello World!<title>
 </head>
 <body>
  <p>Hello World!</p>
 </body>
</html>

This program would make the title of the webpage "Hello World!", and print "Hello World!" on the page.

Example Program in CSS

body {
 background-color:green;
}

p {
 font-family:Impact;
 font-size:26px;
}

This program, when linked to an HTML document, will make the background colour of the page green, and the font of every paragraph on the page, except declared exceptions, have text that is 26 font-size Impact font.
