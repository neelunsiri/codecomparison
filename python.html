Example Program in Python

import time
s=0
min=0
h=0
d=0
while True :
 while h<23:
  while min<59:
   while s<59:
    s=s+1
    print("%s days") %d
    print("%s hours") % h
    print("%s minutes") % min
    print("%s seconds") % s
    print("\n")
    time.sleep(1)
   s = -1
   min = min+1
  h=h+1
 d=d+1

This program is a stopwatch, and prints the time since the program was executed, every second. Its output is in days, hours, minutes, and seconds.
