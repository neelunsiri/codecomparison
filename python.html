Python

Background Information

  • Python is a general-purpose, object-oriented programming language.
  • Python has two stable releases at this point:
    • Python 2.7
    • Python 3
  • Python 2.7 is the most widely used right now.
  • Python 3 is more modern, and adds a few features. The Python Organisation is encouraging people to switch to Python 3, and have limited support for older versions, including Python 2.7.

Example Program in Python

import time
s=0
min=0
h=0
d=0
while True :
 while h<23:
  while min<59:
   while s<59:
    s=s+1
    print("%s days") %d
    print("%s hours") % h
    print("%s minutes") % min
    print("%s seconds") % s
    print("\n")
    time.sleep(1)
   s = -1
   min = min+1
  h=h+1
 d=d+1

This program is a stopwatch, and prints the time since the program was executed, every second. Its output is in days, hours, minutes, and seconds.

Pros and Cons

Pros

  • Very widely used.
  • Very powerful, and has a lot of features.
  • Can be executed straight from the Unix Command Line (Including OS X and Linux).
  • Good for programs that are primarily run on Unix-based Operating Systems (OS X and most Linux distributions).

Cons

  • Windows needs additional software to run Python.
  • Can be difficult to learn for some people.
